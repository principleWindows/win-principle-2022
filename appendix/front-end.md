
# Front-end Frameworks

三大框架: React, vue, Angular 激烈竞争中, 其中 React 处于领先地位。

考虑到微软对 React 的支持力度最大(在 https://github.com/microsoft 中搜索三大框架, 
返回项大大多于另两个), 并且 https://github.com/microsoft/react-native-windows 目前
处于最活跃的开发状态, 可以预言三大框架中 React 对于 Windows 程序员来说未来会得到的官方
更多的技术支撑，使得技术进化有更大的发展空间。
**建议本课程的学习者在目前阶段优先选择前端框架 React Native**:<br>
<https://reactnative.dev/><br>
<https://microsoft.github.io/react-native-windows/>
